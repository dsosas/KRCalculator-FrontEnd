# Environment

Typescript 2.4.2  
Angular 5.0.1  

# Instructions

If have not followed instructions at https://github.com/dsosas/KRCalculator-BackEnd.git please do so first.  

`git clone https://github.com/dsosas/KRCalculator-FrontEnd.git`  
`cd KRCalculator-FrontEnd`  
`npm install`    
`ng serve`
